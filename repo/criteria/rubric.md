# Критерії оцінювання навчальних робіт

## Технічні вимоги

| Вимога | Деталь |
|:-------|:-------|
| Мінімальна кількість слів | 100 слів |
| Дозволені шрифти | Verdana, Arial |
| Дозволені розміри шрифту | 12 пт, 14 пт |
| Використання довгих тире | Заборонено (—, –) |

## Критерії оцінки змісту (1–5 балів кожен)

### 1. Відповідність темі
- **5** — Робота повністю відповідає темі, всі ключові аспекти розкриті
- **4** — Робота здебільшого відповідає темі, незначні відхилення
- **3** — Часткова відповідність темі, є відхилення
- **2** — Слабка відповідність темі, значні відхилення
- **1** — Робота не відповідає темі

### 2. Глибина думки
- **5** — Глибокий аналіз, власні висновки, критичне мислення
- **4** — Достатній аналіз, є власні міркування
- **3** — Поверховий аналіз, переважно переказ
- **2** — Мінімальний аналіз
- **1** — Відсутність аналізу

### 3. Структура
- **5** — Чітка структура: вступ, основна частина, висновки
- **4** — Хороша структура з незначними недоліками
- **3** — Структура є, але з помітними недоліками
- **2** — Слабка структура
- **1** — Відсутність структури

### 4. Оригінальність
- **5** — Яскраво виражена власна позиція, унікальні ідеї
- **4** — Є власна позиція, є оригінальні думки
- **3** — Часткова оригінальність
- **2** — Переважно шаблонний текст
- **1** — Відсутність оригінальності, можливе копіювання

### 5. Якість мови
- **5** — Бездоганна мова, стилістично довершена
- **4** — Хороша мова, незначні помилки
- **3** — Задовільна мова, є помилки
- **2** — Багато мовних помилок
- **1** — Дуже слабка мова

## Шкала підсумкових балів

| Оцінка | Діапазон | Опис |
|:-------|:--------:|:-----|
| А (відмінно) | 90–100 | Видатна робота |
| Б (добре) | 75–89 | Хороша робота |
| В (добре, з зауваженнями) | 60–74 | Прийнятна робота |
| Г (задовільно) | 45–59 | Слабка робота |
| Д (незадовільно) | 0–44 | Незадовільна робота |

> Загальний бал = round(сума_балів / 25 * 100)
